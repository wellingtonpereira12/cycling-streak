# Cycling Streak App

App de acompanhamento de ofensiva de ciclismo com sistema de streak.

## Tecnologias

- **Frontend**: React Native (Expo)
- **Backend**: Node.js + Express
- **Database**: PostgreSQL

## Como Rodar Localmente

### Backend
```bash
cd backend
npm install
node setupDb.js
node server.js
```

### Frontend
```bash
npm install
npx expo start
```

## Deploy

Backend hospedado no Render.com
